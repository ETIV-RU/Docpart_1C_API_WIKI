# Получение информации об артикуле 

Метод возвращает список производителей и наименования товара по его артикулу. Получение данных выполняется с сервера Кроссов Docpart. Для одного артикула может быть возвращено несколько вариантов производителей.

Операция: **ПолучитьИнформациюОбАртикуле**

Параметры запроса:

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|ПараметрыЗапроса *|Структура|Структура отбора для получения заказов|
|НастройкиПодключения|Структура, Ссылка на элемент справочника имНастройкиПодключенияСайтов|Настройки подключения к сайту|


где Параметры запроса - структура с полями: 

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|Артикул *|Строка|Артикул для получения списка возможных производителей|



# Получение списка категорий каталога

Метод возвращает список категорий каталога. Если передано значение параметра parent_id, то будут возвращены только категории, для которых категория с идентфиикатором parent_id является родительской. 

Операция: **КаталогПолучитьКатегории**

Параметры запроса:

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|ПараметрыЗапроса *|Структура|Структура отбора для получения заказов|
|НастройкиПодключения|Структура, Ссылка на элемент справочника имНастройкиПодключенияСайтов|Настройки подключения к сайту|


где Параметры запроса - структура с полями: 

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|ИдентификаторКатегории|Число|Идентификатор родительской категории, необзятельно|


# Добавление новой категории каталога

Операция создает новые категории каталога. Можно создавать нексколько категорий, передав массив в параметре data.

Операция: **КаталогДобавитьКатегории**

Параметры запроса:

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|ПараметрыЗапроса *|Структура|Структура отбора для получения заказов|
|НастройкиПодключения|Структура, Ссылка на элемент справочника имНастройкиПодключенияСайтов|Настройки подключения к сайту|


где Параметры запроса - структура с полями: 

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|ТаблицаДанных|Таблица значений|Таблица с данными категорий для создания на сайте|


где ТаблицаДанных - Таблица с полями: 

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|caption|Строка|Заголовок категории|
|alias|Строка|Представление категории для формирования ссылки на страницу|
|parent|Число|Иентификатор родительской категории|
|title_tag|Строка|SEO Title|
|description_tag|Строка|SEO Description|
|keywords_tag|Строка|SEO Keywords|
|robots_tag|Строка|SEO Robots|
|image|Строка||
|order|Строка|Порядок коатегории|
|published_flag|Число|Признак публикации (1 - опубликовано,0 - не опубликовано)|
|properties|Массив|Массив свойств категории|

где properties - Массив структур с полями: 

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|name|Строка|Имя свойства|
|type|Строка|Тип свойства|
|order|Строка|Порядок отображения в списке|
|list_id|Строка|Идентификатор списка значений свойства|

Параметры запроса:

|**Параметр** |**Пример заполнения** |**Описание** |

Возвращаемое значение: Таблица с полями: 

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|caption|Строка|Заголовок категории|
|alias|Строка|Представление категории для формирования ссылки на страницу|

|keywords_tag|Строка|SEO Keywords|
|robots_tag|Строка|SEO Robots|
|content|Строка|Описание товара|
|published_flag|Число|Признак публикации (1 - опубликовано,0 - не опубликовано)|
|properties|Массив|Массив свойств категории|

где properties - Массив структур с полями: 

|**Параметр** |**Пример заполнения** |**Описание** |
|---|---|---|
|name|Строка|Имя свойства|
|type|Строка|Тип свойства|
|order|Строка|Порядок отображения в списке|
